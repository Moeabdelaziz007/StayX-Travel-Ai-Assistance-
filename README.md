## ✨ Features | المميزات

### 🎙️ AI Voice Assistant — المساعد الصوتي

> Powered by **Gemini 2.0 Flash Live API** — the most advanced conversational AI available.

- **Real-time Voice Interaction** — Talk naturally, get instant intelligent responses
  تفاعل صوتي فوري — تكلم بشكل طبيعي واحصل على ردود ذكية لحظية

- **Animated Voice Orb UI** — Visual waveform feedback for listening / thinking / speaking states
  واجهة الكرة الصوتية المتحركة — تغذية راجعة بصرية لحالات الاستماع والتفكير والكلام

- **AI Tool Calling** — Voice commands trigger real actions: weather check, flight search, itinerary generation
  استدعاء الأدوات صوتياً — الأوامر الصوتية تُشغّل إجراءات حقيقية: طقس، رحلات، خطط

- **Groq Acceleration** — Ultra-low latency text processing alongside Gemini
  تسريع Groq — معالجة النصوص بكمون منخفض جداً بجانب Gemini

---

### 📅 Smart Trip Planning — التخطيط الذكي للرحلات

- **AI-Generated Itineraries** — Day-by-day plans with activities, timings & estimated costs
  خطط رحلات مولّدة بالذكاء الاصطناعي — يومية مفصلة بالأنشطة والتوقيتات والتكاليف التقديرية

- **Google Calendar Sync** — Sync your trips and bookings directly to your calendar
  مزامنة Google Calendar — زامن رحلاتك وحجوزاتك مباشرة مع التقويم

- **Smart Notifications** — Custom sound alerts for new bookings and upcoming events
  إشعارات ذكية — تنبيهات صوتية مخصصة للحجوزات الجديدة والأحداث القادمة

- **AI Destination Images** — Powered by **Pollinations AI** for stunning visual previews
  صور الوجهات بالذكاء الاصطناعي — مدعومة بـ Pollinations AI لمعاينات بصرية رائعة

## 🏗️ Architecture | البنية المعمارية

```
StayX
├── Frontend (Next.js 15 App Router)
│   ├── AI Layer (Gemini 2.0 Flash + Groq)

│   └── State (React 19 hooks + Context)

    ├── RestCountries (Country Info)
    ├── Pollinations AI (Images)
    ├── Google Maps (Location)
    └── Stripe (Payments)
```

## 🚀 Getting Started | البدء

### Prerequisites | المتطلبات

```bash
node >= 20.0.0
npm >= 10.0.0
```

### Installation | التثبيت

**1. Clone the repository | استنسخ المستودع**
```bash
git clone https://github.com/Moeabdelaziz007/StayX-Travel-Ai-Assistance-.git
cd StayX-Travel-Ai-Assistance-
```

**2. Install dependencies | ثبّت المكتبات**
```bash
npm install
```

**3. Set up environment variables | إعداد متغيرات البيئة**
```bash
cp .env.example .env.local
# Edit .env.local with your API keys
# عدّل .env.local بمفاتيح API الخاصة بك
```

**4. Run development server | شغّل خادم التطوير**
```bash
npm run dev
```

**5. Open in browser | افتح في المتصفح**
```
http://localhost:3000
```

## ⚙️ Environment Variables | متغيرات البيئة

Create a `.env.local` file in the root directory with the following:

```env
# =============================================
# AI SERVICES | خدمات الذكاء الاصطناعي
# =============================================
NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key
GROQ_API_KEY=your_groq_api_key

# =============================================
# FIREBASE | فايربيز
# =============================================
NEXT_PUBLIC_FIREBASE_API_KEY=your_firebase_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id

# =============================================
# MAPS & LOCATION | الخرائط والموقع
# =============================================
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_google_maps_key

# =============================================
# TRAVEL APIS | واجهات برمجة السفر
# =============================================
TRAVELPAYOUTS_AFFILIATE_ID=your_affiliate_id
TRAVELPAYOUTS_API_TOKEN=your_api_token

# =============================================
# PAYMENTS | المدفوعات
# =============================================
STRIPE_SECRET_KEY=sk_live_your_stripe_secret_key
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_live_your_stripe_publishable_key
STRIPE_WEBHOOK_SECRET=whsec_your_webhook_secret
```

> ⚠️ **Security Warning | تحذير أمني:** Never commit `.env.local` to version control. It's already in `.gitignore`. | لا تُضف `.env.local` أبداً لنظام التحكم بالإصدار. فهو مدرج بالفعل في `.gitignore`.
